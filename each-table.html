<table data-template="app:list" class="table table-striped"> <tbody> <tr data-template="templates:each" data-template-from="results" data-template-to="entry"> <td data-template="app:list-entry"/> </tr> </tbody> </table>
declare %templates:wrap function app:list($node as node(), $model as map(*)) as map(*) { map:entry('results', ('a', 'b', 'c')) }; declare %templates:wrap function app:list-entry($node as node(), $model as map(*)) { $model('entry') };
templates:each within table

The app:list sets a map entry for 'results' to a sequence.

The templates:each iterates over the sequence identitified by data-template-from and places the value into the map entry 'entry' from the data-template-to. The HTML element containing the templates:each is repeated for each entry in the sequence.

The HTML elements within the HTML element containing the templates:each will use the map entry 'entry' from the data-template-to as one can see in the <td data-template="app:list-entry"/>.
