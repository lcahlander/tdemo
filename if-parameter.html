templates:if-parameter

Parameter myparam is set

Parameter myparam is not set

The templates:if-parameter-set conditionally includes its content only if the given request parameter is set and is not empty. The parameter is referred to by the attribute data-template-param.

The templates:if-parameter-unset conditionally includes its content only if the given request parameter is not set or is empty. The parameter is referred to by the attribute data-template-param.
