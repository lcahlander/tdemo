templates:if-model-key-equals

Foo is set to bar.

Foo is not set to bar.

The templates:if-model-key-equals conditionally includes its content only if the given map entry (whose key is identified by the attribute data-template-key) is set and its value does equal the value (identified by the attribute data-template-value).

The templates:unless-model-key-equals conditionally includes its content only if the given map entry (whose key is identified by the attribute data-template-key) is not set or its value does not equal the value (identified by the attribute data-template-value).
